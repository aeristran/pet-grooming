# 🐾 PawPerfect Pet Grooming Management System

A web-based CRUD application designed to streamline operations for a pet grooming business. This system enables efficient management of appointments, customers, pets, services, and employees.

---

## 📌 Project Overview

**PawPerfect** is a full-stack web application built using PHP and MySQL. It provides role-based access for Admins, Employees, and Customers, supporting appointment scheduling, service management, and user interaction in a user-friendly interface.

---

## 🎯 Project Objectives

- Design a relational MySQL database for a pet grooming business
- Develop a dynamic PHP web application supporting full CRUD operations
- Manage appointments, customers, pets, and services efficiently
- Coordinate development using Agile Kanban methodology
- Deliver a secure and user-friendly grooming management system

---

## 🗄️ Database & System Design

### System Features
- User authentication and role management (Admin, Employee, Customer)
- Pet profiles with medical and grooming details
- Appointment scheduling and service tracking
- Employee management and availability tracking

### Key Tables
- **Users** – authentication and roles
- **Customer Profiles** – contact and address info
- **Pets** – pet details, breed, medical notes
- **Appointments** – scheduling and status tracking
- **Services** – grooming services and pricing
- **Employee Profiles** – staff details
- **Employee Availability** – working hours

---

## 🌐 Website Implementation

### Admin Dashboard
- View all appointments

### Appointment Management
